Tanjong Katong路面塌陷：当局将进行独立调查

发布于：2025年7月28日

新加坡Tanjong Katong路段日前发生严重的路面塌陷事件，导致一辆汽车掉入坑中。新加坡建设局（BCA）今天宣布将对此事件展开独立的深入调查，以查明事故的确切原因，并对现有的工程安全规程进行全面检讨。

初步报告显示，该路面塌陷可能与附近正在进行的深层排污隧道系统（DTSS）的工程有关。公用事业局（PUB）已下令暂停该地区所有相关的隧道工程，并进行紧急安全检查。幸运的是，坠入坑中的司机被热心路人和外籍劳工及时救出，仅受轻伤。

交通部长表示，政府非常重视此次事件，将彻查所有可能导致事故的因素，包括施工方法、地质条件以及监督流程等。他强调，确保公共基础设施的安全是首要任务，任何不合规的行为都将受到严厉追究。当局也提醒公众在此期间绕行该路段，并留意最新的交通信息。

此次事件也引发了公众对城市地下工程安全性的广泛关注。专家指出，随着城市化进程的加快，地下空间的开发日益复杂，必须采用更先进的监测技术和更严格的安全标准，以预防类似事件的再次发生。
