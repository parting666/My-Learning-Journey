9.多列布局(columns)

多列布局可以将内容分成多个垂直列，类似于报纸的排版方式。

这是多列布局的示例文本。多列布局非常适合展示长文本内容，如文章、新闻等。它可以让内容更易读，同时有效利用页面空间。

多列布局允许你指定列数、列间距和列之间的分隔线样式。内容会自动在列之间流动，填满一列后自动转到下一列。

这种布局方式在响应式设计中也很有用，可以根据屏幕宽度自动调整列数，提供更好的阅读体验。

10.响应式布局(mediaqueries)

响应式布局会根据设备屏幕尺寸自动调整，在不同设备上提供最佳显示效果（尝试调整浏览器窗口大小查看效果）。

侧边栏

在小屏幕上会显示在顶部，大屏幕上显示在左侧。

主内容区

在小屏幕上会显示在底部，大屏幕上显示在右侧，占据更多空间。
